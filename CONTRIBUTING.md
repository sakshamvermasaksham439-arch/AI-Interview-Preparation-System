## Contributing Setup

Thank you for your interest in contributing to the AI Interview Preparation System. Follow the steps below to set up the project locally and start contributing.

### 1. Clone the Repository

git clone https://github.com/your-username/AI-Interview-Preparation-System.git

cd AI-Interview-Preparation-System

### 2. Create a Virtual Environment

python -m venv venv

### 3. Activate the Virtual Environment

For Windows:

venv\Scripts\activate

For macOS/Linux:

source venv/bin/activate

### 4. Install Required Dependencies

pip install -r requirements.txt

### 5. Run the Application

streamlit run app.py

### 6. Create a New Branch

git checkout -b feature-name

### 7. Make Your Changes

Add new features, fix bugs, improve documentation, or enhance the user interface.

### 8. Commit Your Changes

git add .

git commit -m "Describe your changes"

### 9. Push Changes to GitHub

git push origin feature-name

### 10. Submit a Pull Request

Open a Pull Request on GitHub with a clear description of the changes you have made.

## Contribution Guidelines

- Follow clean and readable coding practices.
- Test your changes before submitting.
- Keep commits meaningful and descriptive.
- Update documentation whenever necessary.
- Ensure that new features do not break existing functionality.

## Reporting Issues

If you discover a bug or have a feature suggestion, please create an issue and provide:

- A clear title and description
- Steps to reproduce the issue
- Expected behavior
- Actual behavior
- Screenshots (if applicable)

Your contributions help improve the project and are greatly appreciated. Thank you for supporting the AI Interview Preparation System! 🚀
