## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please report it responsibly.

### How to Report

- Create a private report describing the issue.
- Include detailed steps to reproduce the vulnerability.
- Provide screenshots, logs, or proof-of-concept code if applicable.
- Explain the potential impact of the vulnerability.

### Response Process

1. The vulnerability report will be reviewed as soon as possible.
2. Acknowledgment of the report will be provided within a reasonable timeframe.
3. Verified vulnerabilities will be fixed and released in a future update.
4. Reporters will be credited for responsible disclosure when appropriate.
