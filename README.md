An intelligent AI-powered Interview Preparation System built using **Python, Machine Learning, Scikit-learn & Streamlit**.

This system helps students:

✅ Analyze technical skills  
✅ Detect weak & strong topics  
✅ Generate interview questions  
✅ Create AI-based learning roadmaps  
✅ Predict interview readiness using Machine Learning

# 🚀 Getting Started

## 📌 Prerequisites

Before running this project, install:

* Python 3.8+
* pip (Python package manager)

---

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/ai-interview-preparation-system.git
```

---

## 2️⃣ Go To Project Folder

```bash
cd ai-interview-preparation-system
```

---

## 3️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Streamlit Application

```bash
streamlit run app.py
```
